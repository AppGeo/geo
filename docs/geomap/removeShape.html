removeShape

return type jQuery
syntax .geomap( "removeShape", Object shape )
usage 
$("#map").geomap( "removeShape", existingShape )

The removeShape method removes a shape you have previously added with the addShape method and then redraws the map. The existing shape can be an object reference that you have held on to or one that you retrieved from the findShape method.

This method returns the original jQuery collection to maintain call chaining.
