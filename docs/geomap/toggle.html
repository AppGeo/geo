toggle

return type undefined
syntax .geomap( "toggle", String serviceId [, Boolean show_or_hide] )
usage 
$("map").geomap( "toggle", "OSM" )
$("map").geomap( "toggle", "OSM", true )

This convenience method toggles or sets the visible property of the requested service's object in the services array.

If the optional boolean value is not supplied, the visibility of the service will be toggled.

It will then refresh the map.
