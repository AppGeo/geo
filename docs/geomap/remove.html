remove

return type jQuery collection
syntax .geomap( "remove", Object shape )
usage 
$("#map").geomap( "remove", existingShape )

The remove method removes a shape you have previously added with the append method and then redraws the map. The existing shape can be an object reference that you have held on to or one that you retrieved from the find method.

The jQuery UI widget factory returns the original jQuery collection to maintain call chaining.
