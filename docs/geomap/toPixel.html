toPixel

return type Array | Array^2 (GeoJSON position)
syntax .geomap( "toPixel", Array mapPosition | Array^2 mapPositions )
usage 
var pixelPos = $("#map").geomap( "toPixel", [ -71.098709, 42.330322 ] )
var pixelPositions = $("#map").geomap( "toPixel", [ [ -71.098709, 42.330322 ], [ -71.072617, 42.351608 ] ] )

The toPixel method takes a single map position (which is an array of two values: x and y or longitude and latitude) or an array of map positions and returns the pixel coordinates for them.

If $.geo.proj is not null, the widget assumes the arguments are geodetic coordinates and converts them before calculating pixel positions. Otherwise, the widget assumes they are already in its internal projection as defined by tilingScheme for tiled map services or bboxMax for shingled (dynamic) map services.
